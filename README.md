# Hybrid AI Chat Platform is a web-based interface that enables users to interact with AI models in both local offline mode and online API-powered mode.

The application supports GGUF-based local AI models for offline inference, allowing users to run AI directly on their machine without relying on cloud services. It also integrates API-based AI services such as OpenAI, enabling users to switch to cloud-powered responses when an API key is provided.

The goal of this project is to provide a flexible AI chat interface that supports both local AI experimentation and cloud-based AI capabilities in a single platform.

Features

Hybrid AI system supporting local and online AI

Local inference using GGUF models

Cloud AI integration via API keys

Clean and responsive web-based chat interface

Easy switching between offline and online AI modes

Use Cases

Running AI models locally for privacy and offline usage

Connecting to cloud AI services when internet access is available

Testing and experimenting with different AI models in a single interface

HTML

CSS

Local AI inference tools
